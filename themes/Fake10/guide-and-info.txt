"A faithful Windows 10 theme"

by Honu
version 6 (4/4/2026)

What's so different about Fake10 compared to similar themes?:
- Made from scratch (for better & for worse), easily modifiable
- Abides by the Windows 10 art style as much as possible (no Fluent icons, no pre-10 icons¹)
- Doesn't show any branding by default (for any Microsoft software)
- Extremely high attention to detail and accuracy
- Extremely simple accent color customization
- NEW: Contains auto-installer shell script
- NEW: Contains both light and dark modes
- Contains brand new Win10-style icon recreations
- Contains resizable Windows 10 cursors (and an extra inverted cursor theme)
- Contains multiple start button icons to choose from
- Supports the XFCE shade and stick buttons
- Supports Qubes OS and Qubes window coloring

¹except for notepad

PLEASE NOTE: If you're aiming for a very accurate Windows 10-like experience, run the installer or
complete the manual installation guide below! Installing the theme correctly makes it way more
satisfying, as it was designed to complement certain setting changes. I've seen a couple people only
move the icons and GTK themes to their respective folders, doing nothing else. I don't want you to
make the same mistake! Of course, you can do whatever you want with your own system, but be advised.

Aside from that, Fake10 is a fantastic choice for minimal XFCE environments- with it being designed
specifically for Arch Linux and Qubes OS. Heavier distributions should work fine, but they contain a
lot of bundled programs which Fake10 might not support well. Be wary. Otherwise, try it out!

║ GUIDED SETUP ║
╚══════════════╝ (~6 minutes)
Run "chmod +x install.sh" in the same directory as the installer
Run "./install.sh"
Answer prompts and follow instructions
Run "xfconf-query -c xfce4-panel -p / -R -r" to reset the XFCE panel if it breaks [TROUBLESHOOTING]

╔═══════════════════════╗
║ QUBES OS MANUAL SETUP ║
╚═══════════════════════╝ (~70 minutes)
NOTE: The install.sh script is compatible with Qubes OS and significantly reduces installation time.
This guide is for people who don't want to review and run the shell script. Analyzing the script for
Dom0 and your VMs is certainly way faster than doing everything manually, but I want to keep this
guide for more paranoid users. Also, the install.sh SHA256 hash is on the download page, but you should
still manually review the script first. Make sure to quickly check the other scripts in ./Scripts/ too.
###### Dom0 #######################################################################################################################################################
Populate and run the following command in Dom0 to copy the theme to Dom0 [DISCOURAGED BY QUBES DEVS]

	qvm-run --pass-io <source-vm-name> "cat ~/Downloads/Fake10-v6.tar.gz" > ~/Desktop/Fake10.tar.gz

Extract the theme by running "tar -xvzf ~/Desktop/Fake10.tar.gz"
Modify the CSS file in your desired theme (./Themes/Fake10/gtk-3.0/gtk.css or ./Themes/Fake10 Light/gtk-3.0/gtk.css) by running "nano gtk.css" in the directory

	.xfce4-panel .tasklist button box image {
		background-color: transparent;
		margin-left: 5px; <----- [CHANGE TO 9PX]
	}
	.xfce4-panel .tasklist button:checked box image,
	.xfce4-panel .tasklist button:hover box image {
		margin-left: 9px; <----- [CHANGE TO 13PX]
	}

Also modify the following line of CSS

	#whiskermenu-window .category-button {
		border: none;
		background-color: transparent;
		min-width: 30px; <----- [CHANGE TO 36PX]
		min-height: 36px;
		-gtk-icon-style: symbolic;
	}

Modify the following line of CSS

	#genmon-8 .genmon_imagebutton {
		margin-right: -5px; <----- [CHANGE TO -11PX]
	}

Install Fake10 like normal
Copy the window borders from ~/Desktop/Fake10/Qubes/Themes/ to ~/.themes/ and apply one in Window Manager
Back up and overwrite the Adwaita cursors in /usr/share/icons/Adwaita/cursors/ with your desired cursor theme
Modify the following line by running "sudo nano /usr/lib/qubes/icon-receiver" to prevent icon tinting [OPTIONAL]

	icon_tinted = icon.tint(color) <----- [CHANGE TO "icon_tinted = icon"]

Change some VM colors to gray if you don't like the colored context menu borders [OPTIONAL]
Change some VM colors to gray to make their system tray icons monochrome and fit better with the theme [OPTIONAL]
Reboot system to fix icons, cursors, fonts, scrollbars, and alignment
###### Other VMs ##################################################################################################################################################
Get a fresh copy of Fake10 (without the Dom0 CSS modifications)
Modify the following line of CSS in your desired theme (./Themes/Fake10/gtk-3.0/gtk.css or ./Themes/Fake10 Light/gtk-3.0/gtk.css)

	tooltip {
		background-color: transparent; <----- [CHANGE TO WHAT THE COMMENT NEXT TO THIS LINE SAYS]
	}

Install the Fake10 theme in the rest of your VMs (setting accent colors is recommended, skip the taskbar section)
For sys-net, change the VM color to gray and overwrite icons in ~/.icons/Fake10/16/ with icons from ./Qubes/<Light or Dark> Icons/Network Manager/
For whonix-gateway, run " qvm-run -u root whonix-gateway-18 'pcmanfm-qt' " to open a file explorer in root
Copy the sdwdate icons from ./Qubes/Dark Icons/ or ./Qubes/Light Icons/ to the Whonix Gateway VM and overwrite the icons in /usr/share/sdwdate-gui/icons/
